##  Професионално представяне

Аз съм амбициозен и дисциплиниран млад разработчик със силен интерес към:

- 🧠 Системно програмиране
- ⚙ Софтуерна архитектура
- 🌐 Full-Stack разработка
- ☁ Cloud & DevOps технологии

Фокусирам се върху писането на чист, поддържаем и мащабируем код, следвайки принципите на Clean Code и добрите практики в софтуерното инженерство.

Моята цел не е просто да пиша код — а да изграждам стабилни системи.
